## Instructions

1. Lire tous les fichiers `chapitres/*/content.md` dans l'ordre des numéros (via `meta.json`).

2. Pour chaque chapitre, produire un résumé structuré (5-10 lignes) couvrant :
   - **Événements majeurs** : ce qui se passe dans le chapitre
   - **Personnages** : qui apparaît, qui évolue, qui disparaît
   - **Lieux** : où se déroule l'action
   - **Révélations** : informations importantes découvertes
   - **Tensions/conflits** : arcs narratifs ouverts ou résolus

3. Afficher le récapitulatif complet sous forme structurée, prêt à être utilisé comme contexte pour la rédaction du prochain chapitre.

4. Ne pas modifier de fichiers — cette skill est en lecture seule.
