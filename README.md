# Résonance

**Une expérience d'écriture collective assistée par IA.**

Résonance est un roman de science-fiction construit chapitre après chapitre par la communauté. Proposez des idées — personnages, lieux, événements, twists — votez pour vos favorites, et regardez l'histoire prendre forme.

> « Quelque chose dort dans les fondations de la galaxie.
> Et l'humanité, occupée à panser ses propres blessures, a oublié de s'en inquiéter. »

## Lire & Participer

**[resonance.yoandev.co](https://resonance.yoandev.co)**

## Comment ça fonctionne

1. **Proposez** — Soumettez vos idées pour le prochain chapitre (personnage, lieu, événement, objet, twist, lore)
2. **Votez** — La communauté vote pour les meilleures propositions
3. **Lisez** — Le chapitre est rédigé avec l'aide de l'IA à partir de vos idées

Chaque chapitre passe par trois phases : propositions → votes → rédaction. L'epub se construit au fil de l'aventure.

## L'univers

Futur lointain. L'Hégémonie Solaire — trente-sept systèmes, deux cents mondes habités — s'est effondrée il y a cinquante-trois ans. Dans la Lisière, les ruines d'une civilisation alien disparue, les Bâtisseurs, semblent encore actives. Et les Éveillés — des humains aux capacités extraordinaires — vivent cachés.

Genre : sci-fi avec une touche de fantasy. Ton : littéraire, contemplatif, poétique et brutal, inspiré d'*Hypérion* (Dan Simmons).

## Stack technique

- **Frontend :** [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com)
- **Backend :** GitHub Issues & Reactions (pas de base de données)
- **Auth :** GitHub OAuth
- **Déploiement :** Netlify
- **Epub :** Pandoc
